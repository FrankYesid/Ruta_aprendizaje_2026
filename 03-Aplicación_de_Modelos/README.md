# Proyectos de Aplicación de Modelos de Machine Learning

Este repositorio contiene varios proyectos de aplicación de modelos de Machine Learning en diferentes dominios, desde la predicción de enfermedades hasta la optimización de rutas de transporte. Cada proyecto aborda un problema específico y presenta soluciones basadas en algoritmos de aprendizaje automático.

## Proyectos

### 1. Predicción de Enfermedades
**Descripción:** Se utilizan datos de salud para predecir condiciones médicas como diabetes o enfermedades cardíacas.
- **Técnicas utilizadas:** Regresión logística, Random Forest, Redes Neuronales.
- **Dataset:** Datos de salud públicos con variables clínicas y de estilo de vida.
- **Objetivo:** Desarrollar un modelo que permita anticipar el riesgo de enfermedades.

### 2. Recomendación de Libros o Películas
**Descripción:** Implementación de un sistema de recomendación basado en preferencias y comportamiento de los usuarios.
- **Técnicas utilizadas:** Filtrado colaborativo, filtrado basado en contenido, modelos híbridos.
- **Dataset:** Bases de datos de valoraciones de usuarios.
- **Objetivo:** Sugerir libros o películas relevantes a cada usuario según sus preferencias previas.

### 3. Optimización de Rutas de Transporte
**Descripción:** Uso de algoritmos de Machine Learning para optimizar rutas logísticas y reducir tiempos de entrega.
- **Técnicas utilizadas:** Algoritmos de optimización, Redes Neuronales Recurrentes (RNN), aprendizaje por refuerzo.
- **Dataset:** Datos de tráfico, coordenadas GPS y tiempos de entrega.
- **Objetivo:** Minimizar costos y tiempos de transporte mediante la optimización de rutas.

### 4. Predicción del Precio de la Vivienda
**Descripción:** Creación de modelos que estiman los precios de viviendas en función de características locales.
- **Técnicas utilizadas:** Regresión lineal, Gradient Boosting, Random Forest.
- **Dataset:** Datos de ventas inmobiliarias con información sobre ubicación, tamaño, número de habitaciones, etc.
- **Objetivo:** Proporcionar estimaciones precisas de precios de viviendas según sus características.

### 5. Modelos de Clasificación para Reciclaje
**Descripción:** Clasificación de imágenes de basura en categorías como plástico, papel y vidrio para mejorar procesos de reciclaje.
- **Técnicas utilizadas:** Redes Neuronales Convolucionales (CNN), aprendizaje supervisado.
- **Dataset:** Imágenes etiquetadas de distintos tipos de desechos.
- **Objetivo:** Implementar un modelo que identifique automáticamente el tipo de residuo en imágenes.

## Uso del Repositorio

1. **Clonar el repositorio:**
   ```sh
   git clone https://github.com/tuusuario/proyectos-ml.git
   ```
2. **Instalar dependencias:**
   ```sh
   pip install -r requirements.txt
   ```
3. **Ejecutar los notebooks en la carpeta `notebooks/` para explorar cada modelo y sus resultados.**
