# Integración con el Mundo Real

## Descripción

Esta unidad reúne proyectos aplicados a problemas reales que combinan análisis de datos, modelado predictivo y trabajo con conjuntos de datos del mundo real. Hasta el momento se han avanzado varios entregables iniciales, incluyendo notebooks de exploración, preprocesamiento, modelado y scripts de entrenamiento.

## Proyectos

- Proyecto 1: Detección de fraude financiero
  - Se trabajó con datos transaccionales y se elaboró un flujo inicial de exploración, entrenamiento y guardado de un modelo.
  - Archivos destacados: [train_model_script.py](./Proyecto_1/train_model_script.py), [notebooks/01_eda.ipynb](./Proyecto_1/notebooks/01_eda.ipynb), [notebooks/02_model_training.ipynb](./Proyecto_1/notebooks/02_model_training.ipynb).

- Proyecto 2: Pronóstico del clima local
  - Se desarrollaron notebooks para carga y EDA, feature engineering, clasificación y forecasting con LSTM.
  - Archivos destacados: [notebooks/01_Carga_y_EDA.ipynb](./Proyecto_2/notebooks/01_Carga_y_EDA.ipynb), [notebooks/04_LSTM_Forecasting.ipynb](./Proyecto_2/notebooks/04_LSTM_Forecasting.ipynb).

- Proyecto 3: Análisis de impacto ambiental
  - Se avanzó en la exploración de datos, preprocesamiento y modelado con enfoque ambiental.
  - Fuente de datos: [WHO Air Quality Database](https://www.who.int/data/gho/data/themes/air-pollution/who-air-quality-database).
  - Archivos destacados: [notebooks/01_exploracion_datos.ipynb](./Proyecto_3/notebook/01_exploracion_datos.ipynb), [notebooks/02_preprocesamiento.ipynb](./Proyecto_3/notebook/02_preprocesamiento.ipynb), [notebooks/03_modelos_hyperparametros_y_shap.ipynb](./Proyecto_3/notebook/03_modelos_hyperparametros_y_shap.ipynb).

- Proyecto 4: Optimización de cultivos
  - Se ha dejado la estructura base para trabajar con datos agrícolas y modelos de predicción de rendimiento.

- Proyecto 5: Monitorización de transporte público
  - Se prepara la base para analizar rutas, horarios y patrones de movilidad en entornos urbanos.

## Estado general

La unidad se encuentra en una fase inicial de desarrollo, con proyectos bien definidos y con avances concretos en exploración, preprocesamiento y modelado. El siguiente paso consiste en consolidar los resultados en reportes y documentar mejor cada flujo.
